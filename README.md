# Менеджер задач (Task Manager)
Описание проекта
Простое и удобное веб-приложение для управления задачами с возможностью:
Создания, редактирования и удаления задач
Фильтрации задач по статусу, приоритету и категории
Адаптивным интерфейсом для всех устройств
Автосохранением в локальное хранилище (localStorage)

Технологии
Frontend:
React 18
TypeScript
Material UI (MUI) v5
React Router v6
Vite (сборка)

Установка и запуск
Клонировать репозиторий:
git clone https://github.com/your-username/task-manager.git
cd task-manager
Установить зависимости:
npm install
Запустить приложение в режиме разработки:
npm run dev
